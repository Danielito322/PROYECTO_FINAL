# NUBA Reservas · Mobile Premium

Prototipo móvil instalable (PWA) para reservas en **Deportes, Belleza y Entretenimiento** en Puno.

## Abrir
1. Abre la carpeta con VS Code.
2. Clic derecho en `index.html` → **Open with Live Server**.
3. En navegador móvil o modo responsive, se verá como app móvil. En Chrome puedes instalarla desde el icono de instalar.

## Roles
- Cliente: explora, reserva, paga y usa QR.
- Proveedor: administra agenda, productos y valida QR.
- Administrador: revisa negocios, usuarios y reportes.

## Google Maps
La clave solicitada se colocó directamente en `assets/js/config.js` y el mapa se centra en Puno. Antes de subir el repositorio públicamente, restrinjan y roten la clave en Google Cloud.

## Estructura mantenida
Se mantienen todas las páginas originales en `pages/` y se agregaron `login.html`, `cuentas.html` y `admin.html`.
